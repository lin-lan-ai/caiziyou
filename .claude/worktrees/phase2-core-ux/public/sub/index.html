② 打开 Clash → Profiles → 从 URL 下载
  1. 点 Profiles（配置）页面
  2. 在输入框里粘贴刚才复制的链接
  3. 点 下载 按钮

- 如果还是不行，切到「WireGuard App」标签用官方客户端
- 注意：此配置已包含本站直连规则，不会影响你访问菜籽游

📱 WireGuard 官方 App（iOS/Android）

先从下方下载 client.conf，然后按步骤导入：

⬇ 下载 client.conf
1. 打开 WireGuard App
2. 点右下角 + → 从文件或存档导入
3. 找到刚才下载的 client.conf，选中它
4. 导入成功后会看到「纵流-WG」隧道
5. 点右边的开关，开启连接
✅ 开关变绿就说明连上了

🧩 Karing（全平台）

订阅链接方式导入，最方便：

加载中...
1. 打开 Karing App
2. 点击左下角 订阅
3. 点右上角 + → 添加
4. 粘贴刚才复制的链接 → 点 确认
5. 回到首页，点 连接 开关
✅ 连上后状态会显示「已连接」

想手动配置也行，参考 WireGuard App 页面的字段。
